# 📊 E-Commerce Sales Dashboard (Power BI)

## 📌 Project Overview
This project analyzes e-commerce sales data using Power BI to identify sales trends, profit performance, and customer purchasing behavior.

The dashboard provides interactive insights into sales distribution, payment methods, and regional performance.

---

## 🛠 Tools & Technologies
- Power BI
- Data Visualization
- CSV Dataset
- Data Analysis
